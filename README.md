Un administrador de contraseñas simple implementado en Rust

Uso: 
```
password_manager <COMMAND>
```
Comandos:
```
  add     Añade una nueva contraseña
  get     Obtiene la contraseña de un servicio
  list    Lista todos los servicios y usuarios almacenados
  delete  Elimina una contraseña
  help    Print this message or the help of the given subcommand(s)
```
Opciones:
```
  -h, --help     imprime la ayuda
  -V, --version  imprime la version
```
